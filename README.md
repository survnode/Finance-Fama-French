# Finance-Fama-French
Estimating a Famaâ€“French three-factor model that can be used to explain the returns of a diversified portfolio comprising of 30 stocks from different industries. The portfolio is equally weighted consisting of weekly adjusted closing prices for each of the selected stocks over a 5-year period from 15 April, 2017 to 15 April 2022. The model seeks to explain excess portfolio returns by factoring excess market returns (mkt), excess returns on small market capitalization stocks relative to large market capitalization stocks (smb), and excess return on value stocks relative to growth stocks (hml).
